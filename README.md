# Form-React
Um formulário com a lib formik que auxilia muito na criação, e também o uso do yup, que é uma biblioteca para validarmos de forma mais fácil o formulário.

# Tecnologias 
* React⚛️
* Styled components💅
* Lib Formik e Yup ✔️
